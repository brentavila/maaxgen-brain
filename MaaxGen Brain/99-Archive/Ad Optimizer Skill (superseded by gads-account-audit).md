# Ad Optimizer Skill

## Purpose

Analyze ad campaign performance and recommend actions to improve conversions, lower CPA, improve ROAS, and increase lead quality.

## Inputs Needed

- Client brief
- Current offer
- Campaign performance data
- Search terms report
- Keyword report
- Ad copy report
- Landing page URL or notes
- Conversion data
- Change log
- Previous test results

## Process

1. Review the client brief
2. Review the current offer
3. Review the main conversion goal
4. Review campaign performance
5. Identify wasted spend
6. Identify high-performing segments
7. Identify weak keywords
8. Identify weak ads
9. Identify landing page mismatch
10. Review previous changes
11. Avoid repeating failed tests
12. Recommend priority actions
